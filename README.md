# WhatsApp Inspired React Web Application

##  Project Overview
This project is a WhatsApp-inspired web application developed using React.js and Vite. The goal of this project is to recreate a modern messaging interface using component-based architecture while demonstrating responsive design and interactive UI features.

The application simulates a real-world chat system where users can view conversations and send messages dynamically.

---

##  Aim and Objectives
The objectives of this project are:

- To understand and implement React component-based architecture  
- To build a responsive web interface for multiple screen sizes  
- To create reusable UI components  
- To implement basic interactivity such as chat selection and message sending  
- To structure the project using clean and maintainable code practices

##  Component Description

- App.jsx  
  Main component that manages application state and handles chat selection and message updates  

- Sidebar.jsx  
  Displays navigation icons similar to WhatsApp Web  

- ChatList.jsx  
  Displays the list of available chats  

- ChatItem.jsx  
  Reusable component used to render each chat  

- ChatWindow.jsx  
  Displays messages of the selected chat  

- MessageBubble.jsx  
  Reusable component for displaying sent and received messages  

- MessageInput.jsx  
  Input field that allows users to type and send messages
